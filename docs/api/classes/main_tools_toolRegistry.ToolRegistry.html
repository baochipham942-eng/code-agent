    Class ToolRegistry

    Tool Registry - 工具注册表

    管理所有可用工具的注册、查询和代际过滤。 支持 8 代工具的渐进式注册。

    核心功能：

    • 工具注册和注销
    • 按代际过滤工具
    • 云端工具元数据合并
    • 工具定义导出（供模型调用）

    Example

    const registry = new ToolRegistry();

    // 获取 Gen4 可用的所有工具
    const tools = registry.getForGeneration('gen4');

    // 获取特定工具
    const bash = registry.get('bash');

    // 注册自定义工具
    registry.register(myCustomTool);
