## Deferred / Accepted Limitations (by design, not bugs)
- **Cache does not snapshot the external world** (file contents, web, tool results). Resumable = "skip done work, assume environment unchanged" — identical to Claude Code Workflow. A cache hit returns the prior result without re-execution; that is the feature, not a defect.
- **Determinism denylist is binding-unaware and form-specific.** Alias evasion (`const d = Date; d.now()`), `Reflect.apply(Date.now, ...)`, `Date.now.call()`, and shadowing `globalThis`/`global` are NOT caught. Threat model = semi-trusted model code (catch accidental non-determinism, not an adversary) — same tier accepted since P4-A.
- **`run:start` emit is fatal** (run rejects + activeRuns cleared) by deliberate contract; only post-outcome/advisory emits are best-effort.
- **Hard sandbox boundary remains `isolated-vm`** (out of scope; tracked separately).

## Convergence Analysis
Clean convergence over 4 rounds: 6 substantive findings (round 1) → 3 (round 2) → 1 (round 3) → 0 (round 4), each round smaller. **The dominant class across rounds was symmetric application** — round 1 fixed one path, rounds 2–3 found the unprotected siblings:
- HIGH#3 (journal/terminal-emit best-effort) spawned MED#3 round-2 (warning emit) and the round-3 MED (run:done/run:error emit) — the same "observability must not reB execution" principle applied to progressively more emit sites.
- MED#1 round-1 (denylist rewrite) spawned MED#1 round-2 (its own false-positive regression) and MED#2 round-2 (TaggedTemplate sibling shape).

Lesson for next time: when hardening an emit/journal boundary OR rewriting an AST matcher, enumerate **all** call sites / node shapes in the first pass — every emit (start/phase/log/done/error), every journal touchpoint, and Call/New/TaggedTemplate/optional-call AST forms — rather than fixing the one the finding named.
