# Agent Neo Windows 测试版 — 安装指引（发给朋友用）

> 这是小范围测试包，没有购买企业代码签名证书，所以 Windows 会弹两道拦截提示——
> **这是预期行为，不是病毒**。按下面步骤点两下就能装。装完后的自动更新不会再弹任何提示。

## 系统要求

- Windows 10 / 11，64 位（x64）
- 不需要管理员权限（装到你自己的用户目录）

## 安装步骤

1. 下载 `Agent-Neo-<版本号>-win-x64-TEST-setup.exe`（我私发给你的链接或文件）。
2. 双击运行。如果出现蓝色的 **「Windows 已保护你的电脑」**：
   - 点 **「更多信息」**（小字链接）
   - 点 **「仍要运行」**
3. 如果 Windows Defender 弹出拦截：
   - 打开「Windows 安全中心」→「病毒和威胁防护」→「保护历史记录」
   - 找到刚才的拦截记录 → 选 **「允许在设备上」**，再重新运行安装包
4. 安装全程**不会**弹管理员授权（UAC）窗口——如果弹了，截图告诉我。
5. 装完桌面/开始菜单会有 Agent Neo 图标，打开即用。

## 常见问题

- **为什么会被 SmartScreen 拦？** 包没有买证书签名（测试阶段不值当），Windows 对
  所有无签名的新程序都这样提示。
- **以后每次更新都要这么点吗？** 不用。拦截只发生在浏览器下载的首次安装；
  之后 app 内自动更新走的是程序内下载，不经过 SmartScreen。
- **卸载**：设置 → 应用 → Agent Neo → 卸载，和普通软件一样。
