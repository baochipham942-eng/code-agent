# Agent Neo v0.16.100

发布日期：2026-06-09

## 主要变更

### 经验沉淀重做（ADR-020）
- 废弃并物理移除 telemetry n-gram 频次蒸馏路：连续工具调用按频次拼名（如 `bash-bash-bash-bash`）的垃圾 skill 草稿不再产生。
- skill 自动沉淀统一收口到对话语义复盘（conversationReview），升级到 Hermes/Anthropic 规格：任务完成且非平凡（≥2 种语义不同工具或多步数据流依赖）才进入，由 LLM 反思抽取"可陈述的任务意图"决定是否沉淀；抽不出有意义意图则静默不提议。
- 草稿命名走动名词 + 领域宾语（如 `deploying-tauri-macos`），禁用泛词（helper/utils/tools 等）与工具名拼接，在解析与入队两处拦截；产物为结构化 SKILL.md。

### Telemetry 可诊断性
- trace/session 打版本指纹：`agentVersion`（app 版本）+ `promptVersion`（提示词粗粒度标签）+ `toolSchemaVersion`（工具集 schema 短 hash），支持按版本归因失败率。
- 本地全量诊断旁表（`telemetry_raw_payloads`）存全量 prompt/completion/工具入出参，三重封顶滚动淘汰；命中失败信号自动组装自包含诊断包，上传前跑密钥/token/PII 脱敏（本地原文不动，只脱敏上传副本）。
- Langfuse 默认开启，隐私设置页提供一键关闭（opt-out）开关。
- LogMasker 对超大输入预截断，修复脱敏环节 ~110s 卡顿。

### 删除/卸载与权限
- 删除/卸载这类操作直接调用工具，确认交给权限卡片，不再"光说不做"反复说"正在等你确认"。
- 命令安全分级松绑误杀：目标明确的单路径删除（如 `rm -rf /Applications/Xxx.app`）从硬毙降为一次确认；删根/家目录/通配（`rm -rf /`、`~`、`/*`）仍然硬毙。
- 修复挂起权限请求死锁：新消息或取消时能 resolve 挂起请求，不再冻结到 60 秒超时。

### 模型与稳定性
- 强化 provider 选择与诊断，截图发给非视觉模型时不再丢图（改用配置的识图模型）。
- 云端同步会话改幂等 upsert，修复 NULL-owner 主键冲突刷屏。
- sseStream 增加响应头首字节超时，修复 accept-then-hang 干等。
- skill 名称容错兜底解析 + did-you-mean 建议。

## 已知限制（推广前需处理）
- Langfuse 默认链路当前包含内容（dogfood 期接受），正式推广前须收敛为 metadata-only 或服务端代理签发短期 token。
- 失败 session 诊断包当前为静默自动上传（已先脱敏），正式推广前须改回知情同意 + 内联轻提示。
