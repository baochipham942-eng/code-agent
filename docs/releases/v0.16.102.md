# Agent Neo v0.16.102

发布日期：2026-06-10

## 主要变更

### 会话导出修复（打包态静默失败）
- 「导出会话日志」「导出 Markdown」改为主进程直写「下载」文件夹 + 访达自动定位文件，重名自动加 -N 后缀。
- 此前打包态依赖 webview 另存为对话框，在 WKWebView 中会静默失败（点击无任何反应，Intel 实测复现）。
- 导出成功/失败均有 toast 提示，不再存在无反馈路径。

### 其他
- 修复 conversationRuntime 测试 mock 缺失 PROMPT_VERSION 导出导致的 16 个用例失败。

- arm64 / x64 自动更新无感升级。
