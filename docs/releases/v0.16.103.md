# Agent Neo v0.16.103

发布日期：2026-06-11

## 主要变更

### 模型配置修复
- **本地模型不再"假性可用"**：Local(Ollama) 此前不探测本地服务就显示模型可用，没装 Ollama 的机器也会看到本地模型；现在列表展示前先探测本地端点，未运行的服务不再误导。
- **配好 provider 后默认模型自动接管**：新配置 provider 后不再出现"明明配置了还提示默认模型未配置 key"。
- MiMo 托管 key 登录后自动下发，全新安装无需手动配 key 即可体验。

### Windows 测试版（win32-x64）首次随版发布
- 新增 Windows 安装包（NSIS，perUser 安装无 UAC 弹窗），随本版起进入正式发版链；小范围朋友测试用，安装时 SmartScreen 点「更多信息 → 仍要运行」。
- 下载页按访问设备自动推荐对应安装包（Windows 访客置顶 Windows 包，mac 按芯片高亮），所有平台入口仍可见可点。
- Windows 端自动更新链（minisign 校验）随本版起可用。

### 更新分发修复
- 修复更新接口在特定资产排序下可能把配置清单（JSON）当成安装包下发的隐患；Intel mac 在降级路径下不再误取 arm64 包。

## 升级说明
- macOS arm64 / x64 自动更新无感升级。
- Windows 为首个随版测试包，已知降级项（语音输入/OCR/日历邮件等显示"平台不支持"）属预期行为。
