# Agent Neo v0.17.1

发布日期：2026-06-17

## 主要变更

### 发版修复
- **修复 renderer hot-update release gate**：补齐预算设置页新增的 shell capability 声明，避免热更新 manifest 构建把 `getBudgetStatus` / `setBudgetConfig` 误判为当前 shell 不支持。

### 运行账本与恢复能力
- **事件账本主线落地**：权限决策、工具执行生命周期、会话复盘和 Swarm 运行过程进入 append-only ledger，可用于诊断、恢复和对账。
- **崩溃恢复更可信**：新增从执行生命周期重建在飞现场的恢复服务，并暴露 recovery / session ledger / swarm reconcile 等诊断出口。
- **Swarm 降级更稳**：Swarm rollup 可从 ledger 重建，读侧能在 ledger 和旧 rollup 之间 fail-safe 回退，减少半套账本被误当真理源的风险。

### 会话体验与工具反馈
- **会话区降噪**：侧栏、composer、路由 chip、模型选择说明和错误展示进一步收敛，减少引擎内幕和重复状态对正常聊天的干扰。
- **工具错误更可处理**：搜索源耗尽、Bash 长输出、工具失败和 schema parse error 会给出更清楚的摘要、可复制错误和从失败点重试入口。
- **聊天内容更可靠**：修复工具 JSON 被模型回显、auto-load 重试污染会话、假编辑造成上下文双份等问题。

### 预算、权限与安全门
- **预算告警上线**：新增预算设置、运行时预算状态 IPC、StatusBar 用量染色和逼近/超限 toast。
- **权限原因更结构化**：权限 reason 枚举化，并保留面向用户的兼容文案。
- **静态质量门扩展**：新增 console / a11y / stale-dist 扫描脚本，并接入 CI 作为低成本早期信号。

### 设计系统与设置页
- **设计系统契约成形**：新增设计系统文档、静态门和 baseline，开始把颜色、弹窗和组件形态纳入可检查规则。
- **Modal primitive 迁移**：DirectoryPicker、Export、CaptureAdd、Plan、Rewind 等弹窗迁到统一 Modal primitive，真居中弹窗数量继续下降。

### 模型、语音与产品能力
- **模型策略可见化**：模型决策、项目/会话组织、MCP/Skill/插件推荐、prompt stack summary 和 provider 选择展示进一步补齐。
- **语音与热键增强**：端到端语音输入、可配置热键、隐私边界和通知策略进入主流程。
- **能力证据更硬**：新增能力运行证据门、judge 校准数据管道和 eval harness，减少“功能声称完成但缺少运行证据”的情况。

## 修复
- 修复语音转写失败路径绕过隐私防火墙的问题。
- 修复热键裸键和焦点门控问题。
- 修复 shell capability 边界问题。
- 修复预算服务启动期没有用持久化配置初始化的问题。
- 修复会话状态语义，让历史会话不再被错误标红为待处理。
- 修复 renderer 轮询、重复请求和空草稿会话模型继承导致的体验与性能问题。
