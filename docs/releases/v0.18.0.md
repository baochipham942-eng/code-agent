# Agent Neo v0.18.0

发布日期：2026-06-21

## 主要变更

### 设计模式（Design Mode）全量上线
顶层「设计」工作区正式成型，与「通用/Code」工作区并列，覆盖三条产品路径：**交互原型（HTML）+ 设计稿/信息图（无限画布）+ 设计质量自检**。本版本基于对 OpenDesign / Lovart 两款竞品的借鉴调研与多模型对抗评审落地（清单见 `docs/competitive/opendesign-lovart-借鉴清单.md`）。

- **无限画布出图**：konva 自研画布，通义万相文生图直连出图，圈选标注 → 真 mask 局部重绘（wanx2.1-imageedit）。
- **交互原型**：agent 编排单文件 HTML，设备切换、圈选改、版本快照与回滚。
- **设计质量自检**：对 agent 写出的前端源码做确定性 linter（AI 痕迹 + 品味），advisory 回注让模型自我修正。

### 设计模式六项能力增强（T1–T6）
- **变体版本底座（T1）**：canvas 与 proto 统一为非破坏性 Variant——每次操作落新版本永不覆盖，淘汰为软删，支持并排对比与设主版。
- **成本透明 + 可逆历史（T2）**：出图前预估花费，历史每步可命名、可一键回滚，BYOK 实际花费可见。
- **扩图与去水印（T3）**：通义万相 `expand`（四向按比例外扩）与 `remove_watermark`，产物挂入版本底座。
- **一致性锁定再编辑（T4）**：局部重绘后用 region-lock + diff-gate 守住未选区域逐像素不变（越界自动贴回原图并产出 diff 证据图）。
- **方向卡 + 参考截图入口（T5）**：生成前强制澄清表单与多方向选择卡，支持"贴参考截图匹配风格"与"直接生成"逃生口。
- **运行时换肤 + 真图占位（T6）**：原型预览支持 5 套主题色板即时切换（零重生成），配图改用确定性真实图片占位替代灰框。

## 修复
- 修复新增 renderer IPC 能力（设计画布出图/重绘/导入）未登记 shell 能力清单导致的热更新能力闸问题。
- 修复设计图 handler 路径越界守卫（家族级 assertWithinDesignDir 覆盖全部设计 action）。
- 多轮对抗审计修复变体底座、扩图去水印、一致性闸的对称应用与边界问题。
