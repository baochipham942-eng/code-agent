# Agent Neo v0.19.1

发布日期：2026-06-22

## 修复

- **修复 v0.19.0 打包后启动失败**（"Web server exited before healthcheck completed: exit status: 1"）。设计模式的 PDF / PPTX 导出在启动阶段加载 `pdfkit` / `pptxgenjs`，但这两个依赖未随应用打包，导致后端 Web 服务一启动即崩溃、应用无法打开。现已将二者打包进后端 bundle（`pdfkit` 走字体内联的 standalone 构建），打包应用恢复正常启动。

> 此版本仅含上述启动修复，功能与 v0.19.0 一致。从 v0.19.0 更新失败的用户升级到本版本即可正常使用。
