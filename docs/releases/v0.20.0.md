# Agent Neo v0.20.0

发布日期：2026-06-22

## 新增

### 设计模式 — Tab 按交付媒介重规划

设计 tab 按「你想做什么」分 4 类：**网页 / 图 / 演示稿 / 视频**。进来先选媒介，更贴合「我要做一个 ___」的真实心智（`DesignOutputType` UI 聚合，对现有能力零破坏）。

### 设计模式 — 厚版演示稿

需求 →（可 AI）生成大纲 → 逐页编辑（改标题/要点/增删排序）→ 像素预览（LibreOffice 渲染真排版）→（可 AI 配图，模型在页面自选）→ 真排版 PPTX（自动套品牌色）导出。引擎抽成 `slidesGenerator`（SlideData[] 单一真源）；纯文字免费，AI 大纲/配图等增强 opt-in 且成本前置。

### 设计模式 — 参考图垫图

生成前在画布贴入**参考图**（「添加参考图」按钮，画布 sky 徽章区分），生成时首张参考图作为视觉参考喂给模型（通义万相 `wanx2.1-imageedit` / `description_edit`）——保留参考图版式、按需求重新上色出图。

### 设计模式 — 统一历史

设计历史收口到**左侧一处**：图像/视频的步骤时间线（参考图单独分组、不占版本序号），原型的版本看/对比/定稿从预览工具栏并入左侧。图像与原型共用同一套非破坏性版本模型。

## 修复

- 参考图读取失败时显式报错，不再静默退化成纯文生图（避免用户误以为用了参考图）；参考图不显示局部重绘工具栏；续编时清理残留的版本对比状态。

> 本版本合并两批设计模式增强（Tab 4 媒介重规划 + 厚版演示稿，PR #260；参考图垫图 + 统一历史，PR #262），均已通过单元测试、对抗审计与真 key dogfood 验证。
