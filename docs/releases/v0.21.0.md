# Agent Neo v0.21.0

2026-06-27

## 设计画布会自己动手了

**让 AI 直接改画布**：现在不用自己一笔笔改，AI 会把要改的地方先以浅色预览给你看，你点头才落地。

**定个预算，AI 给你几版挑**：设一个变体数量和花费上限，AI 在范围内自己发散出多版，你只管挑喜欢的，超预算会自动停。

**生图模型自己选、视频更顺手**：可以在设置里接入自己的生图模型，画布会避开没配好的；视频能自动配封面、自动适配画面，大视频也能就地播放了。

## 产物做完自动帮你验

**生成的网页/图/演示稿会自动质检**：先做确定性检查，再用视觉判断主观效果，发现问题还能自动尝试修复。

**预览更真实**：网页产物直接在内置浏览器里跑给你看，PPT 改成逐页真实排版截图预览。

## 搜索更聪明

**问得准、源可配**：联网搜索会先规划查询、把更可信的来源排前面，并标注时效性；多个搜索源可以自己启停和排优先级。

## 看得见、花得明白

**看到 AI 派出去的小助手在干什么**：新增 agent 协作视图，子任务的进展一目了然。

**成本日历**：按天 / 周 / 月汇总你的花费，用了多少一眼看清。

## 更稳了

修复了侧边栏列表刷新闪烁、启动瞬间白屏闪、大视频放不动等问题，日常使用更顺手。
