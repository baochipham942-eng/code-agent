# Agent Neo v0.21.1

2026-06-27

## 会话列表清爽多了

**历史会话一眼看懂**：左侧会话列表重做了——每条只显标题和时间，正在跑的显转圈、需要你处理的显小圆点，不再被一堆标签塞满。

**会话太多自动折叠**：同一项目下会话多时，默认只展开前几条，其余收进「展开全部」，长列表不再淹没侧栏。

## 不会再误触花钱

**改掉一个会重复扣费的快捷键**：原来 Cmd/Ctrl+Shift+R 和浏览器「刷新」撞键，手一抖就重发上一条、对出图出视频等于又花一次钱。现在默认不绑这个键，需要可以自己在设置里绑定。

## 更稳更安全

修了画布在非设计场景下被误写的问题，日常使用更稳。
