# Agent Neo v0.23.0

2026-07-01

**在聊天里 @Neo，把一件事交给它**：@Neo 后它会先给你一张「工作卡」讲清要做什么，你确认了再动手，做完在项目协作页一起看产物。整个过程它只会在你批准的范围内干活，不会自己乱动你的东西。

**新增项目协作页**：从侧边栏就能进，集中看 Neo 帮你推进的工作和产出。

**视频生成模型更多了**：除了原有的通义万相和海螺，新增 Veo、Seedance 可选（需对应账号）。

**能出图/出视频/出音乐的模型自动接进来**：你在聊天里配好的模型，只要它支持生成图片、视频或音乐，就会自动出现在多模态页直接能用，不用再单独配一遍。

**新增音乐生成**：可以直接生成音乐了。
