# Agent Neo v0.24.0

2026-07-06

**@Neo 更适合长期项目了**：@Neo 的工作卡更轻，跨会话 topic 续接更顺，之前一件事做了一半，后面可以更自然地接着推进。

**评测和目标闸口更硬了**：GAIA 外部锚点、goal 三闸、产物可运行断言、坏信号回流成回归用例都接进来，Neo 更容易区分是真没做对，还是验证环境自己出了问题。

**上下文和成本更透明**：请求前缀更稳定，活跃工具结果会做轻量裁剪，token / cache-aware 成本和压缩收益能更清楚地归因。

**桌面启动和界面更安静**：窗口首帧等待、renderer ready、侧边栏徽标、设置分组和多处 i18n 都做了修整，启动闪烁、重叠和中英混排会少很多。

**生成和预览链路补了一批稳定性**：游戏预览、导出 metadata、转录渲染、会话工作目录恢复、assistant 消息落库和 Tauri helper 打包路径都做了修复。
