# Agent Neo v0.24.2

2026-07-08

**Mac 安装包改成标准拖拽安装**：双击 DMG 后会看到 `Agent Neo.app` 和 `Applications`，把 app 拖进去再启动，避免用户直接从挂载盘里运行导致下次又像重新安装、重新信任。

**强杀后再打开更稳**：桌面壳启动前会清理残留的旧后台服务端口，避免上一次强制退出留下的 webServer 抢占端口，让下一次启动误判失败。

**发布校验补上安装布局检查**：macOS release gate 会实际挂载 DMG，确认里面有 `Agent Neo.app` 和指向 `/Applications` 的快捷方式，防止再发出不符合安装预期的包。
