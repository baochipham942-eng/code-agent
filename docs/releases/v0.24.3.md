# Agent Neo v0.24.3

2026-07-08

**Claude Code 会复用本机已登录状态**：如果本地 Claude CLI 已经登录，Neo 启动 Claude Code engine 时会继承这份登录态，同时用 safe mode 避开用户 hooks / plugins 对运行结果的干扰。

**`gpt-5.5` 模型不会再因为温度参数直接失败**：这类模型只支持默认温度 `1`，Neo 现在会在运行时自动使用该值；设置页也会锁定温度滑杆并解释原因。

**模型失败提示更像产品提示**：Azure / LiteLLM 的 `temperature` 不兼容、fallback 未配置这类错误会显示成可行动的中文说明，不再把底层路由长串直接糊到聊天里。

**Mac 安装包窗口更整洁**：DMG 继续保留标准拖拽到 Applications 的安装布局，同时调整 Finder 窗口、图标大小和 app / Applications 图标位置。
