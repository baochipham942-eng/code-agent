# Agent Neo v0.24.4

2026-07-09

**模型配置页更像设置页了**：新增 Provider 的入口更明显，执行引擎默认模型也放回设置页解释，不再塞在聊天底部的模型选择器里。

**执行引擎模型支持本机发现**：Codex / Claude 的可选模型会优先从本机 CLI 发现，发现不到时再回到内置目录；Claude Fable、Haiku 这类 alias 也能进入设置页。

**卸载 Ollama 后不再显示旧本地模型**：Local/Ollama 不再因为出厂 `enabled` 或历史模型快照出现在聊天模型弹窗里。只有当前本机服务可达、并且发现到模型时，Local 分组才会展示。

**控制面配置刷新更安静**：并发 cloud config refresh 会合并，减少重复拉取和 renderer bundle telemetry 解析噪音。
