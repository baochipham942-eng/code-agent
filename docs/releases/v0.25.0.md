# Agent Neo v0.25.0

2026-07-11

**Computer Use 终于有连续的共驾指针了**：执行桌面或浏览器操作时，面板内光标会平滑跟随；原生桌面任务还能显示穿透式系统光晕。多显示器、负坐标、混合缩放和跨屏移动都纳入了回归。

**停止、断线和恢复更可信**：Native Run、工具状态、Agent Team 和流式快照现在按 Session / Run 隔离。旧任务的迟到终态不能覆盖新任务，停止后也不会继续出现新的工具执行。

**长会话阅读位置更稳定**：向上加载历史消息时会保留当前视口，用户上滚、搜索定位和持续流式输出各自有独立判断，减少跳动和被强制拉回底部。

**Provider 支持有证据门了**：新增 Provider × Runtime 能力矩阵、请求形状 fixture 和脱敏 smoke ledger。被标记为支持的能力需要自动测试与真实渠道证据，缺证据会在发版时明确阻断或降级。

**权限和执行脚手架更贴合任务风险**：新增只读探索档位，并按模型能力选择修复脚手架密度；无人值守场景继续保持 fail-closed。

**已知边界**：系统级光晕依赖原生 Computer Use 能力确认；不可用时会保留面板内指针，不会伪装成系统级跟随。部分 Provider 的 live smoke 仍受外部额度与渠道状态影响，矩阵会明确显示 `unverified` 或降级状态。
