# Agent Neo v0.25.1

2026-07-11

**登录失败现在会说清楚原因**：Neo 会从认证服务的嵌套响应中提取可行动提示；主动退出后的短暂窗口也会被正确识别，不再误报“会话已过期”。

**生产控制面能更平滑地扛住 Supabase 瞬时故障**：关键读取增加超时、缓存、陈旧数据回退与断路保护。未认证请求不会为了共享密钥额外查询数据库，失败时仍然保持关闭。

**运行数据目录口径已经统一**：运行时和排障文档都指向 `~/.code-agent/code-agent.db`，减少安装版问题定位时查错数据库的风险。

**版本定位**：这是 v0.25.0 的稳定性补丁，不新增产品入口，也不改变连续共驾指针与系统光晕的功能范围。
