# Agent Neo v0.27.2

2026-07-15

- **指哪改哪，改不准会直说**：在预览里指着某一处说要改，现在会精确落到那一处，表格不会再改错行列。万一它定位不到你指的位置，会明确告诉你改不了，而不是默默改到别的地方去。
- **演示稿预览开箱即用**：做 PPT、看多页预览不用再自己额外装东西，装好应用就能用。
- **做演示稿、表格、文档时，参考的例子终于对得上**：以前不管你要做什么，它心里的范例都是写代码；现在你要做演示稿，它就照着演示稿的样子来。
- **要东西就先给东西**：让它做方案、PPT、表格，会先把骨架搭出来给你看，不再一上来就查资料让你干等。
- **改表格更准**：给表格做修改时能直接看到真实行号，不用再靠一行行数。
- **新装的版本能正常收到更新了**。
