# Agent Neo v0.28.0

2026-07-21

- **浏览器和桌面任务现在能在会话里看懂了**：执行到哪一步、正在判断什么、用了哪张截图、产物在哪里，都在同一条时间线上展示；截图会明确区分“已截取”“已分析”和“已验证”，不再把截到图当成任务已经做对。
- **浏览器默认隔离，复用登录态必须逐项授权**：普通任务使用独立 Managed profile。只有你明确授权具体 tab、域名、动作和有效时间后，Relay 才能借用当前登录态；任务结束、停止或授权到期都会归还 tab，不能顺手操作其他页面。
- **浏览器与电脑共用一套控制**：可以暂停、继续、人工接管、停止和结束。停止后旧动作不能继续改页面或桌面，迟到的截图、引用和结果也不能覆盖新一轮执行。
- **复杂任务可以连续完成并留下证据**：支持多个 Surface Session 并发隔离、浏览器与桌面切换，以及“生成 → 浏览器打开 → 截图读取 → 判断 → 调整 → 复验 → 交付产物”的完整工作链；中断后仍能从会话、replay 和导出证据恢复上下文。
- **安全边界更明确**：跨 Agent 访问、过期元素引用、越权域名或动作会直接被拒绝；cookie、profile、输入内容、剪贴板、token 和截图上下文不会原样进入会话、日志、proof 或导出文件。旧 tool 名、历史消息、replay 和 Session export 仍可继续读取。

## 当前边界

- 真实外部账号、验证码、MFA、支付确认和站点反自动化仍需要用户参与；本版不会绕过这些保护。
- 跨站 OOPIF 在没有专用 CDP session 时会 fail-closed；Relay clipboard 与 download 仍保持关闭，Managed 提供已验证的安全路径。
- Relay 扩展的商店签名、上一版本升级与回滚尚未随本次桌面版发布完成，继续使用受控安装与协议版本拒绝边界。
- Windows/Linux Computer provider、外部 Agent transport、企业策略持久化、远程设备池和可携带截图二进制包仍在后续决策门内，本版不会把已验证 seam 宣称为生产能力。
